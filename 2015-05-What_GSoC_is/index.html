What is Google Summer of Code?


"Google Summer of Code is a program designed to encourage university student participation in open source software development."



Organization gain:

  • New contributors and Code

  • Global Exposure


Students gain:

  • Skills

  • Real World Experience

  • Sample Code

  • Contacts


How does Google Summer of Code work?



  1. Mentoring organizations submit applications to Google


  2. List of accepted mentoring organizations is published


  3. The students send their proposals.


  4. Mentoring organizations review and rank student proposals


  5. List of accepted students proposals is published


  6. Students begin coding for their Google Summer of Code projects under advising and evaluation of their mentors.


Tips for students



  1. Start to collaborate in an Open Source project as soon as possible


  2. Quality over quantity


  3. Don't be shy and discuss about the projects, ideas, etc with the organization members


  4. English is the official language during the program, but you don't worry if your english skills are not the best. Code is an universal language.


  5. If it's not publish it doesn't exist. Show the best of you on your website, github, launchpad, g+, bitbucket, etc.
